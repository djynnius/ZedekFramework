Object Relational Mapper

Zedek Framework comes with its own in built Object Relational Mapper class called ZORM. ZORM has a gentle learning curve and allows an Object Relational approach to regular RDBMS like MySQL and SQLite3. It has spun off into its own project called Udara.

Themeing Engine

Our framework allows includes a themeing engine based off the class ZView. This class allows for an almost comlete separation of logic from presentation. Views are typically written in plain html with support for looping! Dynamic views written in PHP are also a feature.

Model-View-Controller

Clean URLs are a feature of the Zedek Framework. The framework is built to allow development of appications with url structure mirroring the MVC. Typically http://app/controller/method with the method often reflecting the view.
